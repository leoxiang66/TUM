change of variables formula

motivation

  • We assume that the data xxx follows a probability distribution p(x)\mathrm{p}(\boldsymbol{x})p(x) i.e.
p(x) where x=[x1⋮xD]\mathrm{p}(\boldsymbol{x}) \text { where } \boldsymbol{x}=\left[\begin{array}{c} x_{1} \\ \vdots \\ x_{D} \end{array}\right]p(x) where x=⎣⎡​x1​⋮xD​​⎦⎤​
  • We can do two interesting things with a distribution:
Data sampling: generate data sample xi\boldsymbol{x}_{i}xi​ following the distribution p(x)\mathrm{p}(\boldsymbol{x}) p(x)
Density evaluation: given any xi\boldsymbol{x}_{i}xi​, compute the probability density at this point p(xi)\mathrm{p}\left(\boldsymbol{x}_{i}\right)p(xi​)
  • Normalizing Flows (NF) can model flexible distributions for data sampling and density evaluation.
  • Normalizing Flows intuition:
    Model a complex distribution by applying a transformation on a simple distribution


idea

  • Normalizing Flows are based on the change of variables formula
  • It transforms a variable z\boldsymbol{z}z into another variable x\boldsymbol{x}x by via a transformation fff i.e. f(z)=xf(z)=\boldsymbol{x}f(z)=x
  • It is particularly useful to simplify computations when working with distributions (or integrals)

The name: "Normalizing flow"

  • A NF transforms a simple distribution (e.g. uniform, Gaussian) into a complex distribution. For some Normalizing Flows the universality theorem has been proven.
  • NFs stack valid transformations to model a complex mapping between the input and output space. The input variable flows through the transformations.
  • The change of variable formula allows to compute the distribution of the output space based on the distribution in the input space. The determinant terms normalize the distribution in the output space. (so that the distribution integrates to 1 —>valid distribution)

Change of variables formula

Change of variables formula (General case): if D∈N,p1(z)D \in \mathbb{N}, \mathrm{p}_{1}(\mathbf{z})D∈N,p1​(z) a DDD-dimensional distribution, f(z)=xf(\mathbf{z})=\boldsymbol{x}f(z)=x an invertible and differentiable transformation, then distribution p2(x)p_{2}(\boldsymbol{x})p2​(x) is
p2(x)=p1(f−1(x))⋅∣det⁡(∂f−1(x)∂x)∣p_{2}(\boldsymbol{x})=p_{1}\left(f^{-1}(\boldsymbol{x})\right) \cdot\left|\operatorname{det}\left(\frac{\partial f^{-1}(\boldsymbol{x})}{\partial \boldsymbol{x}}\right)\right|p2​(x)=p1​(f−1(x))⋅∣∣​det(∂x∂f−1(x)​)∣∣​
  • The determinant term accounts for the distortion扭曲 rate of the transformation (see introductory examples). If it is equal to 1,p2(x)1, p_{2}(\boldsymbol{x})1,p2​(x) and p1(z)p_{1}(\mathbf{z})p1​(z) have the same " volume " at this point i.e. p2(x)=p1(f−1(x))p_{2}(x)=p_{1}\left(f^{-1}(\boldsymbol{x})\right)p2​(x)=p1​(f−1(x)).
    • It considers that the transformation is locally linear (see last example)
  • The term ∂g(x)∂x\frac{\partial g(x)}{\partial x}∂x∂g(x)​ is called Jacobian of ggg; here: a D×DD \times DD×D matrix
    • We have ∂f−1(x)∂x=(∂f(z)∂z)−1\frac{\partial f^{-1}(x)}{\partial x}=\left(\frac{\partial f(z)}{\partial z}\right)^{-1}∂x∂f−1(x)​=(∂z∂f(z)​)−1.
  • The transformation fff should be valid (invertible and differentiable).

Sufficient conditions for a valid transformation fff

  • Invertibility:
    • The input and output space of the mapping should have the same dimension D\boldsymbol{D}D.
    • If D=1\boldsymbol{D}=\mathbf{1}D=1, it is sufficient that fff is strictly monotonic (increasing or decreasing).
    • If the transformation fff is linear, its determinant should be nonzero i.e. det⁡(f)≠0\operatorname{det}(\boldsymbol{f}) \neq \mathbf{0}det(f)=0
  • Differentiability:
    • Both the transformation fff and its inverse f−1f^{-1}f−1 are continuously differentiable, i.e. the Jacobian ∂f−1(x)∂x\frac{\partial f^{-1}(x)}{\partial x}∂x∂f−1(x)​ exists at any point xxx.
    • Note: Differentiability is a sufficient condition; in theory, the mapping fff does not have to be differentiable everywhere, we can even have discontinuous; in practice we usually use only differentiable transformation
